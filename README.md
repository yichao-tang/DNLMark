# DNLMark

This repository contains the code for **DNLMark: Dual Noise Layer-Based Robust Watermarking Against Image Editing**.

## Paper

For more details, see:

Yichao Tang, Mingyang Li, Xinting Wu, Sheng Li, Zhenxing Qian, Xinpeng Zhang*. “DNLMark: Dual Noise Layer-Based Robust Watermarking Against Image Editing.” *IEEE International Conference on Acoustics, Speech and Signal Processing (ICASSP)*, 2026. doi: [10.1109/ICASSP55912.2026.11464215](https://doi.org/10.1109/ICASSP55912.2026.11464215).
